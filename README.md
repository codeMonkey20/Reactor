# Reactor

Environment for creating custom reusable react components for future front end development.
## Installation

Use `git clone` to clone this repository.

```bash
git clone https://github.com/codeMonkey20/Reactor.git
```

After cloning, go to Reactor folder and type `npm install` or `npm i` to install dependencies.
```bash
cd Reactor
npm install
```

Then you are done.
